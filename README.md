## Työkalukauppa

Harjoitustyön aiheena on verkkokauppa, jossa myydään rakennustarvikkeita. Sovelluksen toiminnallisuuksiin kuuluu mm. tuotekategorioiden, tuotteiden ja käyttäjien hallinnointi, sekä tilausten tekeminen.
